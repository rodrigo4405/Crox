# Loxx (.lx)

A simple dynamically typed and object-oriented language.

### Based on Lox by Robert Nystrom. [Link](https://github.com/munificent/craftinginterpreters)
<br />

## Lox++
 Tree-walk interpreter in C++.

 <br />
